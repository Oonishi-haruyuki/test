
# カードクラフター アプリケーション仕様書

## 1. 概要

「カードクラフター」は、ユーザーがAIの力を借りてオリジナルのデジタルカードゲームを作成し、プレイできるWebアプリケーションです。カードのアイデア生成から画像作成、デッキ構築、そして対戦まで、カードゲームの体験全体を包括的に提供します。

## 2. 主要機能一覧

### 2.1. カード作成 (`/`)

-   **AIによるアイデア生成**: ユーザーが入力したテーマ（例：「神秘のドラゴン」）を基に、AI（Genkit）がカード名、能力、フレーバーテキストを自動生成します。
-   **AIによる画像生成**: ユーザーが入力したヒント（英語）を基に、AI（Imagen）がカードのイラストを生成します。
-   **手動カスタマイズ**: カード名、マナコスト、攻撃力、防御力、カードタイプ、レアリティ、能力、フレーバーテキストを自由に編集できます。
-   **ビジュアルテーマ**: カードのフレーム（枠）を複数から選択したり、オリジナルの画像をアップロードして適用したりできます。カード裏面のデザインも選択可能です。
-   **コレクションへの保存**: 作成したカードは、作成コスト（Gコイン）を支払うことで、ユーザーの個人コレクションに保存できます。
-   **PNGエクスポート**: 完成したカード画像をPNGファイルとしてダウンロードできます。

### 2.2. マイカードコレクション (`/collection`)

-   作成・収集したカードの一覧を表示します。
-   カード名での検索、レアリティやマナコストでの並び替え機能を提供します。
-   **カード編集**: コレクション内のカードのテキスト情報（名前、能力、フレーバーテキスト）を編集できます。編集には変更量に応じたGコインが必要です。
-   **カード削除**: コレクションからカードを削除できます。

### 2.3. デッキ構築 (`/deck-builder`)

-   コレクション内のカードを使い、ドラッグ＆ドロップで20枚のカードからなるデッキを構築します。
-   複数のデッキを作成、保存、名前変更、削除できます。
-   **AIデッキ評価**: 構築したデッキをAIが分析し、戦術、長所、弱点、カウンターストラテジーを提案します。

### 2.4. カードガチャ (`/gacha`)

-   Gコインを消費して、ランダムなカードを生成（ガチャを引く）できます。
-   1回と10回のオプションがあり、10回引きには割引が適用されます。
-   生成されたカードはコレクションに保存できます。

### 2.5. AI対戦 (`/battle`)

-   構築したデッキを使用して、AIと対戦します。
-   AIの難易度は「初級」「上級」「超級」から選択できます。
-   AIは難易度に応じて、複数の定義済みデッキ（ゴブリン、エレメンタル、アンデッド等）からランダムに選択されます。
-   勝利すると、ミッションの達成やレーティングの上昇に繋がります。

### 2.6. オンライン対戦 (`/online-battle`)

-   構築したデッキを使用して、他のプレイヤーとリアルタイムで対戦します。
-   **マッチメイキング**: 待機中のプレイヤーを探して自動的にマッチングします。待機中のプレイヤーがいない場合は、新しいゲームセッションを作成して相手を待ちます。
-   対戦データはFirestoreにリアルタイムで同期されます。

### 2.7. ストーリーモード (`/story`)

-    predefinedのシナリオ（章）に沿って、特殊なルール下でAIと対戦するモードです。
-   各章には特定の対戦相手、HP設定、ボードの制限などの独自ルールが設定されています。
-   章をクリアすると、実績やミッションの達成に繋がります。

### 2.8. ショップ (`/shop`)

-   Gコインを使用して、ゲーム内アイテムを購入できます。
-   購入可能なアイテムには、カードフレーム、カード裏面デザイン、進化素材などが含まれます。

### 2.9. カード進化 (`/evolution`)

-   特定のカードと特定の素材（例：「竜の魂」）を組み合わせて、カードをより強力な形態に進化させることができます。
-   進化には成功確率があり、失敗すると素材のみが失われます。

### 2.10. トレード (`/trade`)

-   他のプレイヤーとカードを交換（トレード）する機能です。
-   相手のログインIDを検索し、お互いのコレクションから交換したいカードを選択してオファーを送信します。
-   受信したオファーを承諾または拒否できます。

### 2.11. ギルド (`/guild`)

-   プレイヤーはギルドを設立したり、既存のギルドに参加したりできます。
-   ギルドメンバー専用のリアルタイムチャット機能があります。
-   ギルドメンバーのリストとそれぞれのレーティングを確認できます。

### 2.12. ランキング (`/ranking`)

-   全プレイヤーのレーティングに基づいた上位100名のランキングを表示します。

### 2.13. ミニゲーム (`/minigame`)

-   Gコインを獲得できるミニゲームを提供します。
    -   **ハイ＆ロー**: 次のカードのマナコストが現在のカードより高いか低いかを当てるゲーム。
    -   **マナ当てクイズ**: カードの情報からマナコストを推測するゲーム。
    -   **種族当てクイズ**: カードの情報からカードタイプ（種族）を推測するゲーム。

### 2.14. マイページ (`/mypage`)

-   **認証**: ログインID/パスワード、またはGoogleアカウントでのユーザー認証機能を提供します。
-   **アカウント情報**: ログインID、レーティング、称号などを表示・設定できます。パスワードの変更も可能です。
-   **戦績**: AI対戦の勝利数、敗北数、勝率をグラフで表示します。
-   **実績**: 特定の条件（例：「AI対戦で10回勝利」）を達成すると報酬を獲得し、称号を得ることができます。
-   **ミッション**: デイリー、ウィークリー、マンスリーのミッションを達成して報酬を獲得できます。
-   **対戦履歴**: 過去の対戦のリプレイを再生できます。

## 3. 技術スタック

-   **フレームワーク**: Next.js (App Router)
-   **言語**: TypeScript
-   **UIライブラリ**: React, ShadCN UI
-   **スタイリング**: Tailwind CSS
-   **状態管理**: React Context API
-   **バックエンド/データベース**: Firebase (Firestore, Authentication)
-   **生成AI**: Genkit (Google Generative AI)

## 4. データモデル (Firestore)

-   `/users/{userId}`: ユーザープロフィール（ログインID, レーティング, 称号, 所属ギルドID, インベントリなど）
-   `/games/{gameId}`: オンライン対戦のゲーム状態
-   `/trades/{tradeId}`: ユーザー間のトレードオファー
-   `/guilds/{guildId}`: ギルド情報（名前, メンバーリストなど）
-   `/guilds/{guildId}/messages/{messageId}`: ギルドのチャットメッセージ
-   `/replays/{replayId}`: 対戦のリプレイデータ
