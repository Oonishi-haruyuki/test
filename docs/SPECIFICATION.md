# カードクラフター アプリケーション仕様書

## 1. 概要

「カードクラフター」は、誰でも簡単にオリジナルのデジタルカードゲームを作成し、AIや他のプレイヤーと対戦できるWebアプリケーションです。あなたの創造力をAIがサポートし、カードのアイデア出しから、イラスト作成、デッキの構築、そして熱いバトルまで、カードゲームの全ての体験を一つにまとめました。

## 2. 主要機能一覧

このセクションでは、アプリケーションの各画面で「何ができるのか」を詳しく解説します。

### 2.1. カード作成 (`/`) - あなたの創造力を形に

ゲームの心臓部であるオリジナルカードを作成するページです。

-   **AIによるアイデア生成**:
    -   「AIでアイデアを作成」セクションに、あなたが考えたカードのテーマ（例：「宇宙を旅する侍」「魔法の森のパン屋さん」）を入力して「テキストを生成」ボタンを押すと、AIがそのテーマに合ったカード名、能力、物語を自動で考えてくれます。創作のきっかけに最適です。

-   **AIによる画像生成**:
    -   「画像を用意」セクションで、カードのイラストのイメージを簡単な英語（例：「samurai in space」「magic forest bakery」）で入力し、「AIで画像を生成」ボタンを押すと、AIが世界に一枚だけのイラストを描き上げます。

-   **手動カスタマイズ**:
    -   AIが生成したテキストや、あなた自身が考えたアイデアを基に、カードの全ての項目（名前、マナコスト、攻撃力、防御力、カードの種類、レア度、能力、物語）を自由に編集できます。あなただけの最強カード、面白いカードを作りましょう。
    -   **クリーチャー種族の選択**: カードの種類で「クリーチャー」を選ぶと、その種族を細かく設定できます。選べる種族は以下の通りです。
        -   なし
        -   ヒューマン
        -   エルフ
        -   ドワーフ
        -   ゴブリン
        -   オーク
        -   アンデッド
        -   ドラゴン
        -   ビースト
        -   エレメンタル
        -   ソルジャー
        -   ウィザード
        -   スピリット
        -   天使
        -   悪魔
        -   機械

-   **ビジュアルテーマの選択**:
    -   カードの見た目をカスタマイズできます。ファンタジー風の木枠や、SF風のメタリックなフレームなど、購入したカードフレーム（枠）の中から好きなものを選んだり、自分で用意した画像をアップロードして適用することも可能です。カード裏面のデザインも、ショップで購入したものの中から選んで設定できます。

-   **コレクションへの保存**:
    -   完成したカードは、ゲーム内通貨「Gコイン」を支払うことで、あなたの「マイカードコレクション」に保存できます。保存することで、デッキ構築や対戦で使用できるようになります。作成コストはカードの能力などに応じて変動します。

-   **PNG形式でのエクスポート**:
    -   作成したカードは、画像ファイル（PNG形式）としてコンピュータにダウンロードできます。SNSでシェアしたり、友達に自慢したりしましょう。

### 2.2. マイカードコレクション (`/collection`) - あなたの軌跡

あなたが作成し、集めた全てのカードを管理する場所です。

-   **カード一覧表示**: あなたが今までに集めた全てのカードが一覧で表示されます。
-   **検索と並び替え**: カード名で特定のカードを探したり、「マナコストが高い順」「レア度が高い順」などで並び替えて、コレクションを整理できます。
-   **カードの編集**: コレクション内のカードのテキスト情報（名前、能力、フレーバーテキスト）を編集できます。編集には変更量に応じたGコインが必要です。カードの強さを微調整しましょう。
-   **カードの削除**: 不要になったカードをコレクションから削除できます。

### 2.3. デッキ構築 (`/deck-builder`) - 戦略を練る

コレクションのカードを組み合わせて、対戦で使う「デッキ」を作る場所です。

-   **デッキの構築**: コレクションにあるカードをドラッグ＆ドロップする直感的な操作で、20枚のカードからなるデッキを作成します。
-   **デッキ管理**: 複数のデッキを作成して保存し、それぞれに名前をつけたり、不要になったデッキを削除したりできます。戦術ごとにデッキを使い分けましょう。
-   **AIによるデッキ評価**: 構築したデッキをAIが分析し、「このデッキは速攻が得意」「大きなクリーチャーに弱い」といった戦術、長所、弱点、そしてこのデッキへの対策まで、プロの視点でアドバイスしてくれます。

### 2.4. カードガチャ (`/gacha`) - 運命の一枚を引く

Gコインを消費して、ランダムなカードを生成（ガチャを引く）する機能です。

-   **ガチャを引く**: 「1回引く」と、お得な「10回引く」の2種類があります。何が出るか分からない、ドキドキの瞬間を楽しめます。
-   **コレクションへ保存**: ガチャで手に入れたカードは、あなたのコレクションに自動的に追加されます。思わぬ強力なカードが手に入るかもしれません。

### 2.5. AI対戦 (`/battle`) - 腕試し

自分で構築したデッキを使って、コンピュータ（AI）と対戦するモードです。

-   **難易度選択**: AIの強さを「初級」「上級」「超級」の3段階から選べます。初心者はまず初級から挑戦してみましょう。
-   **多様なAIデッキ**: AIはゴブリン、エレメンタル、アンデッドなど、複数のテーマデッキをランダムに使用してくるため、毎回違った対戦を楽しめます。
-   **勝利報酬**: 対戦に勝利すると、後述するミッションの達成にカウントされたり、オンラインランキングの指標となる「レーティング」が上昇します。

### 2.6. オンライン対戦 (`/online-battle`) - 世界中のライバルと競う

構築したデッキを使い、インターネットを通じて他のプレイヤーとリアルタイムで対戦します。

-   **自動マッチメイキング**: 対戦相手を探している他のプレイヤーを自動で見つけ、マッチングします。もし待機中のプレイヤーがいなければ、あなたが最初の待機者となり、挑戦者を待ちます。
-   **リアルタイム同期**: 対戦中のカードのプレイやライフの変動などは、Firestoreデータベースを通じてリアルタイムに相手の画面に反映され、スムーズなオンライン対戦を実現します。

### 2.7. ストーリーモード (`/story`) - 物語を体験する

あらかじめ用意されたシナリオに沿って、特殊なルールの下でAIと対戦する一人用モードです。

-   **章ごとのシナリオ**: 「ゴブリンの森」「ドラゴンの巣」といった各章には、特定の対戦相手や特別なルール（例：初期HPが低い、特定のカードタイプが使えないなど）が設定されています。
-   **クリア報酬**: 各章の厳しい条件を乗り越えてクリアすると、実績の達成やミッションクリアとなり、特別な報酬が手に入ることがあります。

### 2.8. ショップ (`/shop`) - ゲームを彩る

ゲーム内通貨「Gコイン」を使って、様々なアイテムを購入できる場所です。

-   **購入可能なアイテム**:
    -   **カードフレーム**: カード作成時に使える、豪華なカードの枠。
    -   **カード裏面デザイン**: 対戦中に見える、あなたのカードの裏面デザイン。

### 2.9. トレード (`/trade`) - カードを交換する

他のプレイヤーと、お互いのカードを交換する機能です。

-   **オファーの作成**: 交換したい相手のログインIDを検索し、相手の公開コレクションと自分のコレクションを見比べながら、交換したいカード（あなたが渡すカードと、相手に要求するカード）を選んでトレードの申し込み（オファー）を送信します。
-   **オファーへの応答**: あなたが受け取ったオファーに対して、承諾、拒否、またはキャンセルすることができます。承諾すると、カードの交換が成立します。

### 2.10. ギルド (`/guild`) - 仲間と繋がる

プレイヤー同士でグループ（ギルド）を作り、交流できる機能です。

-   **ギルドの設立と参加**: 自分で新しいギルドを作ることも、既存のギルドに参加することもできます。
-   **ギルドチャット**: ギルドのメンバーだけが参加できるリアルタイムチャットで、情報交換や雑談を楽しめます。
-   **メンバーリスト**: 所属しているメンバーの一覧と、それぞれのレーティングを確認できます。

### 2.11. ランキング (`/ranking`) - トップを目指せ

全プレイヤーの強さの指標である「レーティング」に基づいた、上位100名のランキングを表示します。

### 2.12. ミニゲーム (`/minigame`) - 気軽にコイン稼ぎ

簡単なゲームをクリアして、Gコインを獲得できるコーナーです。

-   **ハイ＆ロー**: 次にめくられるカードのマナコストが、現在のカードより「高い」か「低い」かを当てる運試しゲーム。連続正解で報酬アップ！
-   **マナ当てクイズ**: カードのイラストや名前などの情報から、そのカードのマナコストを推測するクイズゲーム。
-   **種族当てクイズ**: カードの情報から、そのカードが「クリーチャー」なのか「呪文」なのかといったカードタイプ（種族）を当てるクイズゲーム。

### 2.13. マイページ (`/mypage`) - あなたの情報を一括管理

プレイヤー自身の情報を確認・設定するページです。

-   **認証機能**: ログインIDとパスワード、またはGoogleアカウントを使ったログイン・新規登録ができます。
-   **アカウント情報**: あなたのログインID、現在のレーティング、設定している称号などを確認できます。パスワードの変更もここで行えます。
-   **戦績確認**: AI対戦の勝利数、敗北数、そして勝率を分かりやすいグラフで確認できます。
-   **実績システム**: 「AI対戦で10回勝利する」といった特定の条件を達成すると報酬（Gコイン）が手に入り、プロフィールに設定できる「称号」を獲得できます。
-   **ミッション**: 「デイリー（毎日）」「ウィークリー（毎週）」「マンスリー（毎月）」の目標が設定されており、達成するとGコインなどの報酬がもらえます。
-   **対戦履歴**: 過去の対戦の棋譜（リプレイ）を再生し、自分の戦いぶりを振り返ることができます。

## 3. 技術スタック

このアプリケーションは、以下のモダンな技術を使用して構築されています。

-   **フレームワーク**: Next.js (App Router)
-   **言語**: TypeScript
-   **UIライブラリ**: React, ShadCN UI
-   **スタイリング**: Tailwind CSS
-   **状態管理**: React Context API
-   **バックエンド/データベース**: Firebase (Firestore, Authentication)
-   **生成AI**: Genkit (Google Generative AI)

## 4. データモデル (Firestore)

アプリケーションのデータは、GoogleのFirebaseというクラウドデータベースに保存されています。以下はそのデータの構造です。

-   `/users/{userId}`: ユーザー一人一人のプロフィール情報（ログインID、レーティング、称号、所属ギルドIDなど）を保存します。
-   `/games/{gameId}`: オンライン対戦一つ一つのゲーム状態（どちらのターンか、各プレイヤーのライフなど）をリアルタイムで記録します。
-   `/trades/{tradeId}`: プレイヤー間のトレードオファーの内容を保存します。
-   `/guilds/{guildId}`: ギルドの名前や所属メンバーリストなどの情報を保存します。
-   `/guilds/{guildId}/messages/{messageId}`: 各ギルドのチャットのメッセージ履歴を保存します。
-   `/replays/{replayId}`: 対戦のリプレイデータを保存します。
