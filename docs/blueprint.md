# **App Name**: CardCrafter

## Core Features:

- Card Idea Generation: Generates unique card ideas including name, abilities, and flavor text using a generative AI tool. Considers user-specified themes or constraints to tailor the generated card to their card game idea.
- Card Customization: Allows users to customize card details, such as attack power, defense, mana cost, card type, and card rarity.
- Visual Theme Selection: Offers pre-designed visual themes (fantasy, sci-fi, modern, etc.) to apply to the card designs.
- Image Placeholder: Allows users to include placeholders for card images that can be added locally.
- Download: Export card designs in PNG or JPEG format for printing or digital use.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evoking intelligence, strategic gameplay.
- Background color: Light grey (#F0F0F0), provides a clean canvas for card display.
- Accent color: Yellow-amber (#FFC107), highlighting interactive elements, drawing attention.
- Body and headline font: 'Inter', sans-serif, to ensure readability.
- Simple, intuitive icons that reinforce functionality, using a consistent visual language.
- A clean and organized layout that adapts to different screen sizes.
